# Awards Raw Intake

This folder stores raw award collection materials before normalization.

Recommended substructure:

```text
raw/awards/
├─ 2025/
│  ├─ korean-architecture-culture-award/
│  ├─ korean-architecture-exhibition/
│  ├─ young-architect-award/
│  └─ ...
├─ 2024/
└─ 2023/
```

Use this folder for:

- saved HTML
- copied result pages
- PDF notices
- screenshots
- manual extraction notes
- source manifests

The organizing rule here is:

- `year first`
- `award program second`

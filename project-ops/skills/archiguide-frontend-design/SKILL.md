# archiguide-frontend-design

Use this skill for homepage, list pages, and navigation design in `archiguide-kr`.

## Principles

- Follow the structural clarity of architectuurgids.nl.
- Prefer fixed archive navigation, visible counts, restrained typography, and shallow page depth.
- Avoid decorative complexity that hides the archive.
- Use THE B SIDE only as a system reference, never as a content or tone source.

## Deliverables

- clear layout hierarchy
- consistent archive browsing language
- mobile-safe simplified navigation

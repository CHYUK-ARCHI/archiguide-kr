# Data Directory

This directory separates source material, normalized records, and app-ready seeds.

## Structure

- `raw/`: captured source notes, page extracts, spreadsheets, or manual collection dumps
- `normalized/`: cleaned records by entity type
- `seeds/`: compact files intended for use by the application layer

## Current Focus

The first ingestion track starts with:

- Young Architect Award winners
- their offices
- their referenceable projects

Awards are stored as metadata and relations, not as the main browsing system.
